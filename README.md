## Installation

Firstly Fork this repo & then Clone this repo

```bash
  git clone https://github.com/<YOUR_USERNAME>/movies-db-react.git
  cd movies-db-react
```

Install the node_modules

```bash
  npm install
```

run the project

```bash
  npm run start
```
